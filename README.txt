ZCBS LEDSS Session Logger v1.0
================================

Session logging + openBIS upload on END SESSION.

QUICK START
  1. pip install -r requirements.txt
  2. Edit config.yml (URL, space, file types)
  3. Double-click launch.vbs

WORKFLOW
  Data folder + Space → Sign in to openBIS → pick project + experiment
  → START SESSION → save files to folder → END SESSION → upload

See README.md for full documentation.
